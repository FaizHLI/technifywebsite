Past Projects

Olympic Gold Quest (OCQ) (2024)

Olympic Gold Quest is an Indian NGO helping underprivileged athletes in India to pursue Olympic medals. OCQ currently supports around 400 athletes across 10 Olympic and 8 Paralympic sports. This includes around 150 Junior Athletes supported as a part of the OGQ Junior Program and 101 Para Athletes supported as a part of the OGQ Para Program. The project included revamping the website as OGQ's current website is unintegrated between front-end and back-end, as well as building a functional code-based admin panel and database.

Project Kayamanan (2024)

Project Kayamanan aims to raise the financial literacy of public high school students through an online peer financial education in the Philippines. The project included adding responsiveness using media queries (CSS), and improving user information tracking system.

Aquafit (2024)

Aquafit is an Israeli startup providing sexual wellness as well as self-care products and services for women. The project included building a mobile application with different features voiced by Aquafit and prioritizing iterating Aquafit's conversational chatbot into the app.

The Lost Food Project in Malaysia (2021)

Technify partnered with the The Lost Food Project in Malaysia to create a centralized donation application with the aims of onboarding and engaging with more potential food donors by streamlining the process of collecting food donations for neighborhood grocers.

PERWITI Soup Kitchen (2021)

PERWITI is an NGO in Malaysia that feeds the poor and homeless. Technify members had the opportunity to build a web application that recruits and onboards new volunteers while promoting the work of PERWITI soup kitchens in Kuala Lumpur. The web application has live updates on soup kitchen operations for the general public and potential volunteers to view. It also possesses a backend control for PERWITI to register volunteers. The organization is able to feed over 50% of the homeless population in Kuala Lumpur through recruiting more volunteers.
